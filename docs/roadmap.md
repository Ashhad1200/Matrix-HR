# MatrixHR — Roadmap

## MVP Boundary

**Ship in Months 1–6:** Auth, Employee DB, Leave, Attendance, Onboarding, WhatsApp, Self-Service Portal (subset)

**Defer:** Payroll, ATS, Performance, LMS, AI, Marketplace, SSO, custom roles, Meilisearch

## Quarterly Milestones

| Quarter | Milestones |
|---------|------------|
| Q1 (Months 1–3) | Architecture, Auth, Employee DB, Dashboard, App Shell |
| Q2 (Months 4–6) | Leave, Attendance, Onboarding, WhatsApp, MVP launch with 3 pilot clients |
| Q3 (Months 7–9) | Payroll engine, Pakistani tax compliance, bank files, 10 paying clients |
| Q4 (Months 10–12) | Recruitment/ATS, Performance, Self-Service expansion, 30+ clients |
| Q5 (Months 13–15) | LMS, AI features (Ask MatrixHR, resume screening), mobile parity |
