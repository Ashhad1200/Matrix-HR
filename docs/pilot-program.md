# MatrixHR Pilot Program

## Target Profile

- Software houses in Pakistan
- 20–100 employees
- Karachi, Lahore, Islamabad

## Offer

- **Free 3-month pilot** in exchange for:
  - Weekly 30-minute feedback call
  - Permission to publish anonymized case study
  - 2 referral introductions if satisfied

## Onboarding Checklist (per pilot)

1. [ ] Sign pilot LOI
2. [ ] Create tenant subdomain
3. [ ] CSV import all employees
4. [ ] Configure leave policies (Annual 14, Casual 10, Sick 8)
5. [ ] Set flexible 8-hour shift
6. [ ] Assign Software Engineer onboarding template
7. [ ] Connect WhatsApp Business API (optional)
8. [ ] Train HR admin (1-hour session)
9. [ ] Train managers on leave approval (30 min)
10. [ ] Go-live: leave + attendance active

## Success Criteria

| Metric | Target |
|--------|--------|
| Time to first value | Leave + attendance live within 7 days |
| Weekly active usage | 80% of employees clock in weekly |
| WhatsApp adoption | 50% of managers approve via WhatsApp |
| NPS | > 30 |
| Data incidents | Zero critical isolation bugs |

## Weekly Feedback Template

1. What worked well this week?
2. What blocked you?
3. Top 3 bugs or missing features?
4. Would you pay PKR 500/employee/month when payroll ships?

## Conversion Path

- Month 1–3: Free pilot
- Month 4: Offer Growth tier at 20% discount for first year
- Month 6: Full price PKR 500/employee/month
